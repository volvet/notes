# ONNX浅析

## 概述
AlphaGo击败围棋世界冠军李世石以来， 关于人工智能和深度学习的研究呈现井喷之势。 各种新的算法和网络模型层出不穷， 令人眼花缭乱。  与之相随的， 深度学习的训练和推理框架也在不断的退陈出新。 比较著名的有微软的CNTK, Google的Tensorflow， Facebook的PyTorch, Intel的OpenVINO等等.
